# UX Briefing: The UI Fragments into Specialized Workspaces

**April 16, 2026**

Good morning. Today's briefing highlights a clear fragmentation in agent UX. The "one-size-fits-all" chat window is breaking apart into highly specialized interfaces: developer dashboards, unified document workspaces, and deeply integrated data tools. As agents take on more complex, domain-specific tasks, their interfaces are evolving to match.

## Product Highlights

### Claude Code's "Mission Control"

Anthropic has completely overhauled the UX for developers with the redesign of the Claude Code desktop app. [1] They have moved far beyond a simple chat interface, introducing a "Mission Control" sidebar for project management, an integrated terminal, a preview pane for "vibe coding," and a faster diff viewer. Crucially, they added a "Side Chat" feature, allowing users to ask clarifying questions without interrupting the agent's main workflow. 

Furthermore, Anthropic launched "Routines" in research preview. [1] This allows tasks to run autonomously on Anthropic's web infrastructure via schedules or webhooks. This shifts the UX from synchronous interaction (prompt and wait) to asynchronous management (configure and monitor).

### Gemini Merges with NotebookLM

Google has executed a massive UX consolidation by merging Gemini with NotebookLM. [2] This is a significant shift in how users handle complex information. Instead of treating the AI as a conversational partner that occasionally references files, the entire interface is now a unified workspace designed around document organization, research synthesis, and long-term memory. It moves Gemini from a reactive search tool to a proactive research environment.

### Perplexity and Plaid: Data-Driven Agents

Perplexity is demonstrating the power of deep data integration. By integrating with Plaid, users can now connect their live personal financial data directly to Perplexity Computer. [3] This changes the interaction pattern entirely. Instead of asking generic financial questions, users can prompt the agent to perform custom analysis on their actual spending habits and accounts using natural language. The UX challenge here shifts from generating text to securely and accurately visualizing highly sensitive personal data.

## Broader Trends: Visual Collaboration in Chat

The chat interface is becoming increasingly multimodal and collaborative. Lucid Software's integration into ChatGPT via an MCP server app is a prime example. [4] Users are no longer restricted to text-based outputs; they can search, summarize, generate, and share complex visual documents (like flowcharts and diagrams) directly within the ChatGPT interface. The chat window is evolving into a canvas for visual thought.
