## Product Highlights

### Google Gemini: Automating the Android Experience

In a major step towards on-device agency, Google announced on February 25 that **Gemini on Android can now automate multi-step tasks** like ordering food or booking a rideshare. This beta feature, running in a secure virtual window, represents a significant new interaction pattern for mobile users [1]. For developers, Google is introducing **"AppFunctions,"** an early-stage capability that lets apps expose their functionality directly to AI agents. This allows Gemini to perform tasks like retrieving specific photos from the Samsung Gallery without the user needing to open the app themselves [2].

### Perplexity: Introducing the "Computer"

On February 25, Perplexity unveiled **"Perplexity Computer,"** a new system that reframes the agent as a general-purpose digital worker. This marks a significant strategic shift from a chat-based answer engine to an autonomous workflow executor [3].

> With Perplexity Computer, users can describe a desired outcome, and the system will automatically break it down into tasks, create specialized sub-agents, and coordinate their execution. This new interaction model is designed to streamline complex processes like web research, document generation, and data processing from a single natural language prompt [3].

### Microsoft: Secure and Observable UI Automation

Microsoft is heavily investing in the enterprise readiness of its agents. On February 24, it announced enhanced security for its **computer-using agents (CUAs)** in Copilot Studio, including built-in credentials and Azure Key Vault integration for secure, unattended logins [4]. Critically, the platform now offers **advanced monitoring**, including session replay with screenshots and detailed action logs. This provides essential debugging, compliance, and observability for UI automation tasks, a key requirement for enterprise adoption [4]. Separately, an **"Agent Mode"** is coming to PowerPoint, allowing users to create and refine presentations through natural conversation [5].

### Claude: Deeper Enterprise Integration

Anthropic continues to push Claude into specialized professional workflows. On February 24-25, it expanded **Claude Cowork** with new enterprise plugins for finance, HR, and engineering, aiming to boost the productivity of office workers by embedding the agent directly into their specialized software ecosystems [6]. The company also introduced **"Fast mode"** and performance-tier pricing, giving users more control over the trade-off between response speed and cost [11].

### Developer Tooling: Agents in the IDE

The latest VS Code Insiders build (v1.110), announced February 25, introduces **native browser integration for AI agents**. This allows agents to interact directly with web page elements, access the console, and take screenshots for visual debugging, all from within the editor [12]. This development is a crucial step in streamlining web development workflows by keeping agentic capabilities tightly integrated within the IDE, rather than requiring developers to switch contexts. The update also includes improved visibility into sub-agents and context usage, giving developers more control and insight into the agent’s operations [12].
