## Product Highlights

### The Agent Gets a Driver’s License: Claude Controls Your Computer

The most significant UX development today is from **Anthropic: Claude can now operate a computer** [1]. In a new research preview for macOS Pro and Max users, the agent can open applications, browse the web, and complete tasks directly on the user’s machine. This is a landmark step in agentic autonomy. A user can now ask Claude to check emails and generate a report, and the agent will perform these actions itself. While this represents a powerful new interaction model, it also brings user concerns about control and dependency to the forefront.

### The Agent Goes to Work: Microsoft’s Copilot Cowork

Microsoft is all-in on the agent as a productivity partner. The latest wave of **Microsoft 365 Copilot updates deeply embeds the agent into Word, Excel, and PowerPoint** [2]. Users can now create, edit, and refine content with the agent’s help without ever leaving the application. The new **“Copilot Cowork,”** built in partnership with Anthropic, allows for multi-step, delegated work, transforming the agent from a simple assistant into a true collaborator. This is further enhanced by an improved “Work IQ Memory,” which leverages a user’s work history for more personalized and context-aware responses.

### The Agentic Commerce Reckoning

While some platforms are doubling down on agentic capabilities, the path to autonomous commerce is proving to be a rocky one. **OpenAI is reportedly deprioritizing its Instant Checkout feature in ChatGPT**, and **Walmart has shut down its own agentic commerce experiment with OpenAI**, citing significantly lower conversion rates compared to its native website [4]. This serves as a crucial reality check: just because an agent *can* do something doesn’t mean users will want it to. The user experience of trust, reliability, and value remains paramount, and the industry is learning that a seamless checkout flow is a hard UX problem to solve.
