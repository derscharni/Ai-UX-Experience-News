# UX Briefing: The Agent Comes Home

**March 22, 2026**

A fresh, concise overview of the latest user experience developments in the AI agent landscape. Today’s theme: **The Agent Comes Home.** The frontier of agentic AI is moving from the cloud to your local machine. A new class of desktop agents is emerging, with direct access to your files and applications, fundamentally changing the interaction model from a web chat to a true desktop assistant. This is happening alongside a broader industry reckoning, as user feedback begins to rein in the excesses of the “AI everywhere” gold rush.

## Product Highlights

### The Agent Comes Home: A New Desktop Paradigm

The most significant trend this week is the agent’s migration from the cloud to the desktop. **Manus has officially launched its “My Computer” app** for both macOS and Windows [1]. This gives the agent the ability to operate directly on a user’s local machine, interacting with files, browsers, and applications. This is a fundamental shift in the interaction model. Instead of pasting text into a web chat, users can now give the agent access to a folder and a goal, like “organize these thousands of photos into categories.” Critically, the UX requires user approval for every action, providing a crucial layer of trust and control.

Following the same trend, **Perplexity has unveiled its “Personal Computer”**, a new product described as a unified AI system for complex tasks [2]. While details are still emerging, it points to the same conclusion: the future of personal AI is a persistent, dedicated agent that lives on your hardware, not just in a browser tab. A new platform war for the “agent as OS” is officially underway.

### The User Strikes Back: Microsoft’s Copilot Retreat

In a stunning admission that more AI isn’t always better, **Microsoft is rolling back some of its most aggressive Copilot integrations in Windows 11** [3]. Responding directly to user feedback about “AI bloat,” the company is removing Copilot entry points from apps like Photos, Widgets, and Notepad. This is a major course correction and a victory for users who felt the AI was being pushed too aggressively into their workflows. It signals a maturation of the market, where “genuinely useful” is becoming a more important metric than “AI everywhere.”

### Streamlining the Experience

As agents become more powerful, platforms are working to make that power more accessible. **ChatGPT has simplified its model picker to just three options: “Instant,” “Thinking,” and “Pro”** [4]. By sunsetting niche personalities like “Nerdy” and clarifying the tiers of reasoning available, OpenAI is making it easier for users to choose the right tool for the job without being overwhelmed. This focus on a clean, simple UX is a recurring theme, as seen in Claude’s new interactive visualizer [5], which replaces static data tables with dynamic, explorable charts, making data analysis more intuitive and engaging.
