# UX Briefing: The Agentic Workflow

**March 16, 2026**

A fresh, concise overview of the latest user experience developments in the AI agent landscape. Today’s theme: **The Agentic Workflow.** The focus is shifting from single-shot commands to complex, multi-step task execution. New interaction patterns are emerging to manage this complexity, from side-question commands that preserve context to the agent’s ability to use a computer mouse and keyboard just like a human.

## Product Highlights

### Claude Can Now Use a Computer

The most significant new capability this week comes from Anthropic. The new **“Computer Use” feature**, launched with Claude 3.5 Sonnet, gives the agent the ability to operate a computer in a human-like way [1]. It can browse the web, fill out forms, and even order food. This moves beyond simple API integrations and represents a fundamental shift in the interaction model. The agent is no longer just a brain; it’s a digital body that can directly manipulate the web.

However, this new power is not without its challenges. Anthropic’s new AI code reviewer, which leverages similar technology, has already drawn criticism from developers over high token costs and job displacement fears, underscoring the UX and ethical hurdles of deploying highly autonomous agents [8].

### Manus Introduces the Side Conversation

As agents take on longer and more complex tasks, managing the conversation becomes a new UX challenge. Manus has introduced an elegant solution: the **“/btw” (by the way) command** [2]. This allows a user to ask a quick side question without derailing the agent’s primary workflow. The agent takes a snapshot of its current context, answers the tangential query, and then seamlessly resumes its main task. It’s a simple, intuitive pattern for enabling parallel conversations and a powerful tool for managing the cognitive load of interacting with a long-running agent.

### ChatGPT Gets Interactive and Transactional

OpenAI continues to enrich the ChatGPT experience, moving beyond static text responses.

*   New **interactive visuals for math concepts** allow users to manipulate sliders and drag components to see formulas in action, turning the agent into a hands-on learning tool [3].
*   In the UK, financial services company **Experian has launched a credit score application directly inside ChatGPT** [4]. This is a significant step toward making the agent a hub for real-world financial transactions, building on the app integrations launched just a day prior.

### The Community Pushes Back: “Stop Sloppypasta”

A new manifesto-style website called **“Stop Sloppypasta”** has gained significant traction on developer forums [7]. It’s a community-led reaction against the proliferation of low-effort, unedited AI-generated content being dumped into documents, codebases, and team chats. The movement advocates for a set of norms around the responsible use of AI output, emphasizing human review, editing, and clear attribution. It’s the first organized, grassroots effort to establish a culture of quality and accountability in the agentic era — a sign that the social and professional UX of AI is becoming just as important as the product UX.
